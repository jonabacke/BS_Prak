This function takes a task out of a task queue. If the task queue is empty, this function blocks until an element is available.

Parameters

tThe header of the task, that will be send.
argBufA buffer for the argument structure to be send.
sizeOfArgBufSize of argBuf
Returns
In case of an error, the program terminates with an error message. On success 0 will be returned. In case of an non blocking access to a full queue, 1 will be returned.

This function adds a task in a task queue.

Parameters
mqdesThe descriptor of the task queue.
tThe header of the task that should be added.
argPointer to the argument structure of the task.
blockingThis parameter determines whether sendToTaskQueue should block in case of a full task queue.
Returns
On success the return value is 0. If the queue is full and blocking == false, then the task is not inserted in the queue and the return value is 1. In case of an error, an error message will be given an the program terminates.
